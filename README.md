# Sul-ijoa(술이조아)

<br>
점점 주류 가격이 높아져서 밖에서 외식하기 꺼려지는 요즘, 기존의 식당을 알려주는 플랫폼말고 소주가격별로 가게를 추천, 소개하는 플랫폼이 있다면 좋지 않을까?
소주 가격을 우선적으로 미리 알아보고 가격별 부담없이 술먹읍시다!
